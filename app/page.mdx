# Agent Docs

This repo holds documentation for Hermes-based agent deployments and the path from bespoke installs to a full SaaS platform.

## Current Sections

- [Deployment Strategies](deployment-strategies): cloud, dedicated tenant, hybrid local inference, and full on-prem deployment modes.

## Operating Principle

Manual deployment work should become one of three things:

- a reusable internal checklist
- a productized SaaS onboarding feature
- a conscious enterprise-services offering
