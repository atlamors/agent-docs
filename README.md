# Agent Docs

Nextra documentation site for Hermes-based agent architecture, deployments, and operations.

The first section is Deployment Strategies, which compares four deployment topologies:

- Cloud Lite
- Dedicated Cloud Tenant
- Hybrid Local Inference
- Full On-Prem Appliance

## Local Development

```bash
npm install
npm run dev
```

Then open `http://127.0.0.1:3000/agent-docs`.

## Build

```bash
npm run build
```

The project is ready to import into Vercel as a Next.js app.
